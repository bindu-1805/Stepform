# Stepform 

This is a Stepform consisting of multiple pages created with React. <br />
The pages are, 
1. Personal Details
2. Address Details
3. Payment Details
4. Confirmation page

## Getting started 

1. Clone the repository
   ```
   git clone https://github.com/bindu-1805/Stepform.git
   ```
2. Install dependencies (if needed)
   ```
   npm install
   ```
3. Run the app
   ```
   npm start
   ```
